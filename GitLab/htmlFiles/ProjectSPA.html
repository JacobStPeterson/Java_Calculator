Página de ayuda de la calculadora Rimplex

Botones y sus funciones

± : Cambia el número en la calculadora de negativo a positivo.
C : Borra la entrada en la calculadora.
R: Borra la entrada y el historial de la calculadora.
<< : Este botón actúa como retroceso.
Inv : Invierte la entrada en la calculadora.
Dec/Frac : Cambia la entrada de la calculadora a decimal o fracción.
i : Esto permite la implementación de números imaginarios en la calculadora.
Real : Aísla la parte "Real" (verdadero) de un número complejo.
Imag : Aísla la parte "Imaginary" (imaginaria) de un número complejo.
ans : Copie y pegue la respuesta a la entrada anterior.

Símbolos y números básicos

+ : Adición.
- : Sustracción.
X : Multiplicación.
/ : División, también utilizado para fracciones.
^ : Exponentes, permite al usuario agregar un exponente a la entrada.
√ : Raíz cuadrada
ϴ : Theta, utilizado para la forma polar.
. : Punto decimal.
(: Paréntesis izquierdo.
) : Paréntesis derecho.
= : Signo igual.
0-9 : numero 0 - 9 .

Otra Notas

Menú : En la pestaña de menú, puede guardar e imprimir el estado actual de la calculadora.
Ahorrar : Antes de guardarlo, se le pregunta dónde le gustaría guardar su historial y en qué formato le gustaría guardarlo. Después de seleccionar, puede ingresar el nombre del archivo y guardar el archivo. También puede cancelar en cualquier momento.
Impresión : Permite al usuario imprimir en un archivo.
Ajustes : Muestra al usuario una lista de configuraciones para elegir.
Idioma : El usuario puede elegir un idioma para elegir, el predeterminado actual es el inglés.
Ayuda : El menú acerca de y ayuda, acerca de las conversaciones generales sobre el producto y lo más probable es que haya llegado a esta página a través de más ayuda.
